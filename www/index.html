How it works

From voice to clipboard
in seconds.

01 🎙️

Record

Double-tap Ctrl. The tray icon turns red. Speak naturally.

02 🔊

Normalize

ffmpeg normalizes volume and encodes to Opus for optimal upload size.

03 🤖

Transcribe

Groq runs Whisper large-v3-turbo — typically under two seconds.

04 ✍️

Correct

An LLM fixes typos, punctuation, and grammar without changing meaning.

05 📋

Paste

Text is copied to clipboard and pasted into your active window via xdotool.

Features

Everything you need,
nothing you don't.

⌨️ Global hotkey

Double-tap Ctrl captures input from any application — browser, editor, terminal, chat.

🌍 Multi-language

Switch between Czech and English. Whisper auto-detects accent; the LLM corrects grammar accordingly.

🎵 Music auto-pause

Detects playing media via playerctl, pauses it while you dictate, and resumes automatically.

📜 Persistent history

Every transcription is saved to a local JSON file. Browse, compare raw vs. corrected, and copy with one click.

🔁 Translation

Optional one-step translation to English via the same LLM — no additional API needed.

🪟 Qt6 GUI

Native PySide6 interface with system tray, history browser, settings dialog, and status bar.

🔒 Fully local control

Only the audio clip is sent to Groq for transcription. No telemetry, no accounts, no subscriptions.

⚙️ Modular codebase

Clean separation: audio, transcription, clipboard, music, GUI — each in its own module.

Installation

Up and running
in four steps.

01 — System tools
sudo apt install \
  alsa-utils ffmpeg \
  xclip xdotool playerctl
02 — Clone & install
git clone https://github.com/\
  robert-marik/voice-to-text.git
cd voice-to-text
pip install -e .
03 — API key
# Free tier at console.groq.com
export GROQ_API_KEY="gsk_..."

# Make it permanent:
echo 'export GROQ_API_KEY="gsk_..."' \
  >> ~/.bashrc
04 — Run
python main.py

# Look for the blue circle
# in your system tray.
# Double-tap Ctrl to start.
